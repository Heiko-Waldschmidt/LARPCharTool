Features and their Usage

Feature HowTo
A Convention History of a Character can be added and will be saved in a H2 Database.
  1. Add a Character (at "Characters" page)
  2. Add a Convention (at "Conventions" page).
  3. Add a Visit of the Character at the Convention (at "Visits" page).
Cascading deletes: If you delete a Convention or a Character all regarding Visits were automatically deleted.
  • To delete a Character hit on "Delete" at the "Characters" pages.
  • To delete a Convention hit on "Delete at the "Conventions" page.
Planned: Filtering. Goal: See the list of all visits of only one Character.
Planned: Calculation of the sum of Condays per Character.
Planned: Export of the list of the Visits one Character.
